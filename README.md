# ARTI308_Lab4

This lab focuses on data quality assessment and preprocessing using the BMW Car Dataset as part of the ARTI308 Machine Learning course.
The work includes the following tasks:

Data Quality Assessment – Checking data types, structure, and consistency issues in the dataset.
Missing Value Handling – Detecting missing values and applying median imputation.
Outlier Detection & Treatment – Using the Interquartile Range (IQR) method to detect outliers and applying capping to reduce their impact.
Data Normalization – Applying Min-Max scaling and Z-score standardization to ensure features are on comparable scales.
Correlation Analysis – Generating a correlation heatmap to analyze relationships between numerical features.
Dimensionality Reduction (PCA) – Applying Principal Component Analysis to reduce feature dimensions while preserving variance.

All steps are implemented using Pandas, NumPy, Matplotlib, Seaborn, and Scikit-learn to ensure the dataset is clean, consistent, and ready for machine learning modeling.
